NeMo Gym is **a Gymnasium for LLMs** — RL training environments with a familiar `reset()` / `step()` API. This section gets you from zero to a scored rollout, then to a custom environment of your own.

<Tip>
Already installed and have an API key? Jump straight to the [Quickstart](/latest/get-started/quickstart).
</Tip>

## Pick Your Path

| If you want to... | Start here | Best for |
| --- | --- | --- |
| **Understand the architecture first** | [About](/latest/about) → [Concepts](/latest/about/concepts) → [Prerequisites](/latest/get-started/prerequisites) → [Installation](/latest/get-started/installation) → [Configuration](/latest/get-started/configuration) → [Quickstart](/latest/get-started/quickstart) | New to RL for LLMs |
| **Run an evaluation right now** | [Installation](/latest/get-started/installation) → [Configuration](/latest/get-started/configuration) → [Quickstart](/latest/get-started/quickstart) | Comfortable with RL training frameworks |
| **Build a custom environment** | [Installation](/latest/get-started/installation) → [Configuration](/latest/get-started/configuration) → [Create Your First Environment](/latest/get-started/create-first-environment) | Comfortable with agentic RL |

**What you need regardless of path:**

1. Python 3.12+ on a [verified configuration](/latest/get-started/prerequisites#verified-configurations).
2. An OpenAI-compatible inference endpoint and API key (or self-hosted vLLM — refer to [Configuration](/latest/get-started/configuration#do-you-need-an-api-key)).
3. Familiarity with the command line and basic Python.
