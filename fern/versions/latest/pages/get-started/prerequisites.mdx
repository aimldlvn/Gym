What you need to install and run NeMo Gym.

Minimum bar: Python 3.12+, 8 GB RAM, an OpenAI-compatible API key.

## Hardware Requirements

| Component | Minimum | Notes |
| --- | --- | --- |
| **CPU** | Any modern x86_64 or ARM64 | Intel, AMD, or Apple Silicon |
| **RAM** | 8 GB | 16 GB+ recommended for concurrent rollouts |
| **Storage** | 2 GB free | More for benchmark data |
| **GPU** | None | Required only for local vLLM inference or NeMo RL training |

## Software Requirements

| Component | Requirement |
| --- | --- |
| **OS** | Linux (Ubuntu 20.04+), macOS (11.0+), or Windows (WSL2) |
| **Python** | 3.12 or higher |
| **Git** | For cloning the repository (Source install path) |
| **Docker** | For the container install path |
| **API key** | OpenAI, Azure, or self-hosted vLLM endpoint (refer to [Configuration — Do You Need an API Key?](/latest/get-started/configuration#do-you-need-an-api-key)) |

For cluster, Kubernetes, or production deployment specifics, refer to [Deployment Topology](/latest/reference/deployment-topology).
