`ng_reward_profile` prints:

```text
key_metrics:
  mean/reward:  -0.60
  max/reward:    1.00
  min/reward:   -1.00
  mean/total_tokens:  73.8
```

That's GPT-4.1's win rate playing 10 hands of Blackjack against the dealer. The full path from a fresh terminal to that number takes **~10 minutes** and **~$0.05**.

## Prerequisites

1. NeMo Gym installed via the [Source path](/latest/get-started/installation) (`ng_list_benchmarks` is not yet in the PyPI release).
2. `env.yaml` configured with your model credentials ([Configuration](/latest/get-started/configuration)).
3. Virtual environment activated: `source .venv/bin/activate`.

## Quickstart Steps

1. **Discover the CLI**

   ```bash
   ng_help
   ```

   **✅ Success Check**: A table of `ng_*` commands prints. Append `+h=true` to any one of them for detailed usage.

2. **List Available Benchmarks**

   ```bash
   ng_list_benchmarks
   ```

   **✅ Success Check**: A table prints with ~30 benchmarks (`gpqa`, `aime25`, `ifbench`, and others). For this Quickstart, you'll run the smaller, self-contained Blackjack environment to learn the flow.

3. **Start the Servers** (Terminal 1)

   ```bash
   ng_run "+config_paths=[\
   resources_servers/blackjack/configs/blackjack.yaml,\
   responses_api_models/openai_model/configs/openai_model.yaml]"
   ```

   **✅ Success Check**: Four `Uvicorn running on http://127.0.0.1:<port>` lines print — one each for the head, [resources, model, and agent servers](/latest/about/concepts/core-components). Leave this terminal running.

4. **Collect Rollouts** (Terminal 2)

   ```bash
   mkdir -p results
   ng_collect_rollouts \
       +agent_name=blackjack_gymnasium_agent \
       +input_jsonl_fpath=resources_servers/blackjack/data/example.jsonl \
       +output_jsonl_fpath=results/blackjack_rollouts.jsonl \
       +num_repeats=2
   ```

   **✅ Success Check**: `Collecting rollouts: 100%|████| 10/10` prints. Each [rollout](/latest/about/concepts/key-terminology#rollout--data-collection-terms) is one full hand: the model saw its cards, replied `<action>hit</action>` or `<action>stand</action>`, and the env returned reward `+1` (win), `0` (draw), or `-1` (loss/bust).

   <Accordion title="Flag reference">
   | Flag | Meaning |
   |---|---|
   | `+agent_name` | [Agent server](/latest/about/concepts/core-components) to invoke (from `ng_list_benchmarks` or your YAML). |
   | `+input_jsonl_fpath` | Input dataset, one task per line. |
   | `+output_jsonl_fpath` | Output file. Also writes `<output>_materialized_inputs.jsonl`. |
   | `+num_repeats` | Rollouts per task. Higher = stabler metrics. |
   | `+limit` | Cap the number of input tasks (smoke tests). |
   | `+num_samples_in_parallel` | Concurrency. Defaults to unlimited. |
   </Accordion>

5. **Read the Win Rate**

   ```bash
   ng_reward_profile \
       +materialized_inputs_jsonl_fpath=results/blackjack_rollouts_materialized_inputs.jsonl \
       +rollouts_jsonl_fpath=results/blackjack_rollouts.jsonl
   ```

   **✅ Success Check**: The output at the top of this page prints. GPT-4.1 lands around `mean/reward = -0.6` on this 10-rollout sample. A real evaluation uses `+num_repeats=16` or higher for stable numbers.

6. **Stop the Servers**

   In Terminal 1: `Ctrl+C`.

The same three commands — `ng_run`, `ng_collect_rollouts`, `ng_reward_profile` — work for every environment listed by `ng_list_benchmarks`. Only the config paths and dataset change.

## Troubleshooting

<Accordion title="Command not found: ng_run">
The virtual environment is not activated in this terminal:

```bash
source .venv/bin/activate
```
</Accordion>

<Accordion title="Missing mandatory value: policy_api_key">
You have not created `env.yaml` yet. Refer to [Configuration](/latest/get-started/configuration).
</Accordion>

<Accordion title="OpenAI 429: quota exceeded">
Add credits at [platform.openai.com/account/billing](https://platform.openai.com/account/billing). The Quickstart needs ~$0.05.
</Accordion>
