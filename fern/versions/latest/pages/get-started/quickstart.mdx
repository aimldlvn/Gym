# Quickstart

<Info>
This guide assumes NeMo Gym is already installed. If not, see [Installation](/latest/get-started/installation).
</Info>

## CLI

```bash
# Display all available commands
ng_help

# Get detailed help for any command
ng_run +help=true
ng_test +h=true
```

Every CLI command supports `+h=true` or `+help=true` for detailed usage information.

## Run Evaluation

Start your model and agent servers:
```bash
ng_run "+config_paths=[<model_config>,<agent_config>]"
```

Score your agent on your tasks:

```bash
ng_collect_rollouts \
    +agent_name=<agent> \
    +input_jsonl_fpath=<data> \
    +output_jsonl_fpath=<output>
```

Calculate evaluation metrics:

```bash
ng_reward_profile \
    +materialized_inputs_jsonl_fpath=<data> \
    +rollouts_jsonl_fpath=<output>
```
