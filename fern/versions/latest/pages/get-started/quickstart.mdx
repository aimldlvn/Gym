Install NeMo Gym, configure your model credentials, and score an LLM agent at [`blackjack`](https://github.com/NVIDIA-NeMo/Gym/tree/main/resources_servers/blackjack). **Time**: about 10 minutes. **Cost**: about $0.05 in OpenAI API usage.

Each rollout in the output JSONL has a top-level `"reward"` key:

```json
{"reward": 1.0,  ...}   // win
{"reward": 0.0,  ...}   // draw
{"reward": -1.0, ...}   // loss / bust
```

Averaging those over 10 rollouts gives GPT-4.1 around `-0.6` against a fixed dealer.

## Prerequisites

1. Python 3.12+ on a [verified configuration](/latest/get-started/prerequisites#verified-configurations).
2. An OpenAI API key, or another OpenAI-compatible endpoint (Azure, NIM, self-hosted vLLM).

## Quickstart Steps

1. **Install NeMo Gym**

   ```bash
   git clone git@github.com:NVIDIA-NeMo/Gym.git
   cd Gym
   uv venv --python 3.12 && source .venv/bin/activate
   uv sync
   ```

   For PyPI or container install paths, refer to [Installation](/latest/get-started/installation).

2. **Configure Your Model Credentials**

   Create `env.yaml` in the project root:

   ```bash
   cat > env.yaml << 'EOF'
   policy_base_url: https://api.openai.com/v1
   policy_api_key: sk-your-actual-key-here
   policy_model_name: gpt-4.1-2025-04-14
   EOF
   ```

   Replace the placeholder API key with your real key. For Azure, NIM, or self-hosted vLLM, refer to [Configuration — Provider Examples](/latest/get-started/configuration#provider-examples).

3. **Start the Servers** (Terminal 1)

   ```bash
   ng_run "+config_paths=[\
   resources_servers/blackjack/configs/blackjack.yaml,\
   responses_api_models/openai_model/configs/openai_model.yaml]"
   ```

   `ng_run` blocks while the [resources, model, and agent servers](/latest/about/concepts/core-components) run. Four `Uvicorn running on http://127.0.0.1:<port>` lines print — one each for the head, resources, model, and agent servers. Leave Terminal 1 running and open a second terminal.

   <Tip>
   On a repeat run with no dependency changes, append `+skip_venv_if_present=true` to `ng_run` to skip venv re-creation.
   </Tip>

4. **Collect Rollouts** (Terminal 2)

   ```bash
   mkdir -p results
   ng_collect_rollouts \
       +agent_name=blackjack_gymnasium_agent \
       +input_jsonl_fpath=resources_servers/blackjack/data/example.jsonl \
       +output_jsonl_fpath=results/blackjack_rollouts.jsonl \
       +num_repeats=2
   ```

   5 example tasks × 2 repeats = 10 rollouts. Each [rollout](/latest/resources/glossary#rollout-also-trajectory) is one full hand: the model sees its cards, replies `<action>hit</action>` or `<action>stand</action>`, and the env returns reward `+1` (win), `0` (draw), or `-1` (loss / bust).

   <Accordion title="Flag reference">
   | Flag | Meaning |
   |---|---|
   | `+agent_name` | [Agent server](/latest/about/concepts/core-components) to invoke. |
   | `+input_jsonl_fpath` | Input dataset, one task per line. |
   | `+output_jsonl_fpath` | Output file. Also writes `<output>_materialized_inputs.jsonl`. |
   | `+num_repeats` | Rollouts per task. Higher = stabler metrics. |
   | `+limit` | Cap the number of input tasks (smoke tests). |
   | `+num_samples_in_parallel` | Concurrency. Defaults to unlimited. |
   </Accordion>

5. **See the Results** (Terminal 2)

   ```bash
   python3 -c "
   import json
   rewards = [json.loads(l)['reward'] for l in open('results/blackjack_rollouts.jsonl')]
   print(f'mean reward: {sum(rewards)/len(rewards):+.2f} ({len(rewards)} rollouts)')
   "
   ```

   Outputs `mean reward: -0.60 (10 rollouts)` or similar. For a stable evaluation, raise `+num_repeats` to 16 or higher.

6. **Stop the Servers**

   In Terminal 1: `Ctrl+C`.

Blackjack is wired up for fast smoke testing. The same `ng_run` + `ng_collect_rollouts` flow runs any benchmark from `ng_list_benchmarks` (`gpqa_diamond`, `ifbench`, `math_with_judge`, and others) — only the config paths and dataset change.

## Troubleshooting

<Accordion title="Command not found: ng_run">
The virtual environment is not activated in this terminal:

```bash
source .venv/bin/activate
```
</Accordion>

<Accordion title="Missing mandatory value: policy_api_key">
`env.yaml` is missing or in the wrong directory. From the project root:

```bash
ls env.yaml
cat env.yaml | head -3
```

Re-create the file using the snippet in step 2.
</Accordion>

<Accordion title="OpenAI 429: quota exceeded">
Add credits at [platform.openai.com/account/billing](https://platform.openai.com/account/billing). The Quickstart needs about $0.05.
</Accordion>

<Accordion title="ng_collect_rollouts can't reach the servers">
Verify the servers came up healthy with `ng_status`. Expect `3 servers found (3 healthy, 0 unhealthy)`. If any are unhealthy, check Terminal 1 for stack traces.
</Accordion>
