# Getting Started

## Installation

Requires Python 3.12+ on x86_64 or ARM64 (Linux, macOS, Windows via WSL2). No GPU required.

<Tabs>
<Tab title="Git clone">

```bash
git clone git@github.com:NVIDIA-NeMo/Gym.git
cd Gym
uv venv --python 3.12 && source .venv/bin/activate
uv sync
```

</Tab>
<Tab title="pip install">

```bash
python3 -m venv .venv && source .venv/bin/activate
pip install nemo-gym
```

</Tab>
<Tab title="Container">

For RL training with NeMo RL, use the latest NGC container:

```bash
docker pull nvcr.io/nvidia/nemo-rl:v0.5.0.nemotron_3_super
```

See the [NeMo RL training tutorial](/latest/training-tutorials/nemo-rl-grpo/setup) for setup details.

</Tab>
</Tabs>

## CLI

```bash
# Display all available commands
ng_help

# Get detailed help for any command
ng_run +help=true
ng_test +h=true
```

Every CLI command supports `+h=true` or `+help=true` for detailed usage information.

## Run Evaluation

Start your model and agent servers:
```bash
ng_run "+config_paths=[<model_config>,<agent_config>]"
```

Score your agent on your tasks:

```bash
ng_collect_rollouts \
    +agent_name=<agent> \
    +input_jsonl_fpath=<data> \
    +output_jsonl_fpath=<output>
```

Calculate evaluation metrics:

```bash
ng_reward_profile \
    +materialized_inputs_jsonl_fpath=<data> \
    +rollouts_jsonl_fpath=<output>
```
